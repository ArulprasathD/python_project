# 🐍 Hospital Patient Data Analysis Project

## 📌 Project Overview 

This project is based on a **Hospital Patient Dataset** containing **500 patient records**.  
The main purpose of this project is to practice and demonstrate **Python for Data Analysis**, **Data Cleaning**, **Missing Value Handling**, **Visualization**, and **Machine Learning**.

This project is very useful for beginners who are learning:

- 🐍 Python
- 📊 Pandas
- 🔢 NumPy
- 📉 Matplotlib
- 🤖 Scikit-learn

---

# 📂 Dataset Used

**File Name:** `hospital_patient_dataset.csv`

This dataset was created for **practice purposes** and includes patient-related medical details.

---

# 🧾 Dataset Columns

The dataset contains the following columns:

| Column Name | Description |
|------------|-------------|
| `patient_id` | Unique ID for each patient |
| `name` | Patient name |
| `gender` | Gender of the patient |
| `age` | Age of the patient |
| `blood_pressure` | Blood pressure value |
| `sugar_level` | Blood sugar level |
| `cholesterol` | Cholesterol value |
| `weight` | Weight of the patient |
| `doctor_visits` | Number of doctor visits |
| `recovered` | Recovery status (`Yes` / `No`) |

---

The goals of this project are:

- ✅ Read and understand CSV data using Python

- ✅ Perform statistical analysis
